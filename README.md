## Library and Apps
- MongoDB
- Express.js
- React.js
- Node.js
- bcryptjs
- React Google Login
- React router dom

## Run Steps
1. change directory to server and type "npm run start" in terminal.
2. change directory to client and type "npm run start" in terminal.
